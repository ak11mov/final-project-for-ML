# final-project-for-ML
my project was created for delivery and further development and improvements, here you can track the complete creation of the project with comments and a clear interface.

проект создан для моего последнего задания по ML который представляет из себя локальное веб приложение с помощью которого работники банка смогут определять по данным который предоставляет клиент выдавать ему кредит или нет и также какие риски несёт банк выдавая клиенту который допустим не имеет заработка 

данные для обучения модели взяты с kaggle 
- https://www.kaggle.com/datasets/laotse/credit-risk-dataset

Возможности
- вы можете использовать данное приложение для проверки своего кридитного рейтинга (выдадут ли вам его)
- также добавленна рандомная функция в виде рулетки для одобрения кредита без данных
- возможность допилить эту работу и интегрировать сервис в работу банка для упрощения многих процессов
- создание сайта для того чтобы пользователь мог проверить свои щансы на одобрение

 ##  Установка
 
1.Установить архив

2.открыть его с помощью (vscode ,pycharm)

3.запустить файл app.py 

4.В терминале прописать uvicorn app:app --reload

5.Пройти по пути который выдал терминал ПРИМЕР(http://127.0.0.1:8000)

6.Вуаля вы смогли локально развернуть веб прложение 

7.Если по какой то причине вы не хотите открывать через браузер то вы можете октрыть файл script_api.ipynb и там в формате json внести данные и получить предикт
